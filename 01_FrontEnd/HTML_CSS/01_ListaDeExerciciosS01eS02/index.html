5 Hábitos que Todo o Programador Júnior Já Deveria Ter

Aprender a programar é algo realmente sensacional, mas independente se você é um programador júnior ou programador sênior, o ato de estudar e trabalhar com programação pode deixar você louco. Daí você coloca na cabeça que, para melhorar suas habilidades, você deve fazer um curso de programação, seja ele pago ou qualquer outro curso de programação grátis que você encontrar pela internet, e de fato apesar de começar a entender melhor a linguagem de programação ou framework que você usa, em pouco tempo as coisas começam a ficar difíceis de novo, você começa a ficar frustrado com bugs, começa a perder prazos de sprints que prometeu, e com certeza começa a se perguntar se Ciência da Computação é algo para você (e claro que isso vale para qualquer outra formação, como Engenharia da Computação, Engenharia de Software, Sistemas de Informação ou Análise e Desenvolvimento de Sistemas). Se você já passou por isso, esse vídeo é para você, principalmente se você é um iniciante e quer enfrentar o mundo de tecnologia de uma forma mais inteligente. E também se você está procurando como aprender a programar, esse vídeo vai lhe dar a base para isso acontecer de uma forma mais saudável.
